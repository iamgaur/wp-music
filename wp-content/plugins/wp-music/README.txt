WP Music for add music records and show listing

== Description ==

This plugin will give the option to add music content Insert, Update, and Delete from the backend. It's also provide show music listing with help of shortcodes [music]

== Installation ==

via Wordpress
1. Go to your admin area and select Plugins â†’ Add New from the menu.
2. Click on Upload Plugin button
3. Click Choose Now button and select the zip file `wp-music.zip`
4. Click install.
5. Click activate to activate plugin.
